Drop your reel videos here to make them autoplay on the homepage.

Name them: reel-1.mp4, reel-2.mp4, reel-3.mp4, reel-4.mp4, reel-5.mp4, reel-6.mp4
Optional poster images (shown before the video loads): reel-1.jpg, reel-2.jpg, ...

Tips:
- Compress to H.264 MP4, vertical 9:16, ideally under ~5 MB each so the page stays fast.
- The videos autoplay muted and loop when scrolled into view.
- Until a file exists, that card falls back to a tap-to-play Instagram embed,
  so the section never looks broken.

To change which reels show, edit the `reels` array in
src/components/ReelsShowcase.tsx.
